## 🔧 Instalação

Clone o repositório:

```bash
git clone https://github.com/THShikota/php.git
```

Acesse a pasta do projeto:

```bash
cd php
```

---

## ▶️ Como executar

Caso possua o servidor embutido do PHP:

```bash
php -S localhost:8000
```

Depois acesse no navegador:

```text
http://localhost:8000
```

Como alternativa, execute diretamente pelo terminal:

```bash
php index.php
```
