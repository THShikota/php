# PHP

Projeto de exemplo desenvolvido em PHP para demonstrar a execução de um script simples que imprime mensagens na tela.

---

## ✨ Funcionalidades

* Exibição de mensagens utilizando PHP.
* Exemplo básico de execução de código PHP.

---

## 🚀 Tecnologias utilizadas

* PHP

---

## 📦 Estrutura do projeto

```text
.
└── index.php
```

### Descrição

| Arquivo     | Descrição                                                                   |
| ----------- | --------------------------------------------------------------------------- |
| `index.php` | Arquivo principal da aplicação responsável por exibir as mensagens na tela. |

---

## ⚙️ Pré-requisitos

Antes de executar o projeto, é necessário possuir:

* PHP instalado (versão não informada).

---

## 🔧 Instalação

Clone o repositório:

```bash
git clone https://github.com/THShikota/php.git
```

Acesse a pasta do projeto:

```bash
cd php
```

---

## ▶️ Como executar

Caso possua o servidor embutido do PHP:

```bash
php -S localhost:8000
```

Depois acesse no navegador:

```text
http://localhost:8000
```

Como alternativa, execute diretamente pelo terminal:

```bash
php index.php
```
